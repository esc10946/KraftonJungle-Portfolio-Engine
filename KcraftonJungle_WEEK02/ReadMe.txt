추가 기능 설명: 자력과 탄성

1.1. 자력 설명
	- 화면 중앙(0, 0, 0)을 자성의 중심으로 설정하여 모든 공이 중앙으로 끌려오는 기능을 구현했습니다.
	- ImGui를 통해 Magnetism체크박스로 실시간으로 자력의 활성화 여부와 세기를 조절할 수 있습니다.

1.2. 자력 구현
	- 각 프레임마다 공의 현재 위치에서 원점까지의 방향 벡터를 계산합니다.
	- 계산된 방향으로 ImGui로 설정된 자력 세기와 델타 타임(DeltaTime)을 곱한 가속도를 매 프레임마다 속도(Velocity)에 합산합니다.
	- 중력과 자력이 동시에 작용할 때의 복잡한 움직임을 표현하라 수 있습니다. 

2.1. 탄성 설명
	- 다른 공들과 충돌할 때 탄성 계수를 적용하여 계수에 따라 반발력 크기가 달라지도록 구현했습니다.
	- ImGui를 통해 Elasticity를 체크하면 탄성계수 세기를 조절할 수 있고 체크 해제하면 비탄성계수가 적용됩니다. 

2.2. 탄성 구현
	- 두 공 사이의 거리와 법선벡터를 계산하여 부딪히는 순간 질량에 따라 밀리는 거리를 계산하여 겹침을 없앴습니다. 
	-  충돌지점의 법선 벡터를 기준으로 상대속도를 계산하고 충격량 공식을 적용 후 두 공의 Velocity를 충격량과 질량에 따라 반영했습니다.
